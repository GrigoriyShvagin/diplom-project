# Диаграммы (draw.io)

Диаграммы для пояснительной записки и технического отчёта (`../../_report.md`).
Формат — `.drawio` (diagrams.net). Открываются в приложении [draw.io](https://app.diagrams.net),
десктоп-версии или расширении «Draw.io Integration» для VS Code.

Для вставки в Word/пояснительную записку: открыть файл → **File → Export as → PNG/SVG**
(рекомендуется PNG, 300 dpi, с прозрачным фоном или белым).
